# students-data-analysis

# Overview

This dataset contains a large collection of student records used for data analysis and visualization. It is designed to help explore academic performance, demographic distribution, and statistical patterns among students.
The dataset is typically used for practicing:

> Data cleaning
> Exploratory Data Analysis (EDA)
> Statistical analysis
> Data visualization
>

# Technologies Used
🐍 Python Libraries
1. Pandas
Used for: Loading CSV file (read_csv)
Data cleaning and preprocessing
Handling missing values
Grouping and aggregation (groupby)
Filtering and transformation

2. NumPy
Used for: Numerical computations
Statistical calculations (mean, median, std deviation)
Efficient array operations
Handling numeric transformations

3. Matplotlib
Used for: Data visualization
Creating plots such as:
Bar charts (e.g., department-wise performance)
Histograms (e.g., marks distribution)
Line plots (e.g., trends)
Scatter plots (e.g., attendance vs marks)

# Example Analysis Performed
Average marks per department
Gender-wise performance comparison
Distribution of grades
Correlation between attendance and marks
Top-performing students
Outlier detection in scores
